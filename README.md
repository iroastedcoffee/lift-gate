# r1.ieee.org

Static site migrated from [https://r1.ieee.org/northjersey](https://r1.ieee.org/northjersey) using **Liftkit Migrator**.

## Live site

After GitHub Pages finishes building, the site will be available at:

```
https://iroastedcoffee.github.io/lift-gate/
```

## Local preview

Just open `index.html` in your browser, or serve the directory:

```bash
npx serve .
```

## Structure

- `index.html` — homepage with post listing
- `posts/*.html` — individual blog posts
- `pages/*.html` — WordPress pages
- `assets/styles.css` — single stylesheet
